# 项目功能总览

## 项目定位

Aric's Blog 是一个前后端分离的个人博客与内容管理系统。
它同时覆盖博客展示、内容发布、效率工具、AI 辅助、后台管理和部署运维。

## 用户角色

| 角色 | 能力 |
| --- | --- |
| 游客 | 浏览首页、文章、分类、标签、归档、相册、说说、友链、留言、关于页 |
| 登录用户 | 评论、留言、个人中心、使用效率工具、参与互动 |
| 管理员 | 管理文章、分类、标签、评论、留言、用户、菜单、角色、日志、任务和站点配置 |

## 核心能力

- 内容展示：首页、文章详情、分类、标签、归档、搜索、相册、图床、友链、留言板、关于页。
- 内容管理：文章、页面、分类、标签、说说、友链、评论、留言、相册、照片、轮播。
- 用户与权限：登录、第三方登录、验证码、后台菜单、角色权限、在线用户。
- 效率工具：待办、任务池、日历、甘特图、日记、思考沉淀、时间块、习惯追踪、AI 助手。
- AI 能力：搜索页 AI 对话、效率工具 AI 总结与建议、提示词管理。
- 运维能力：Docker Compose 部署、Nginx 反代、MySQL、Redis、文件存储、日志和任务。

## 系统边界

| 范围 | 说明 |
| --- | --- |
| 博客前台 | 面向访客和登录用户 |
| 后台管理 | 面向管理员 |
| 后端服务 | 提供 API、认证、业务逻辑、权限、文件、日志、任务 |
| 部署配置 | 容器、反向代理、环境变量、初始化脚本 |
