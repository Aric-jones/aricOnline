# 模块功能说明

本文档用于维护各模块职责和功能边界。

## 顶层模块

| 模块 | 路径 | 职责 |
| --- | --- | --- |
| 博客前台 | `blog-vue/shoka-blog` | 博客展示、互动、搜索、效率工具、AI 助手 |
| 后台管理 | `blog-vue/shoka-admin` | 内容管理、系统管理、日志、任务、监控 |
| 后端服务 | `blog-springboot` | 业务 API、认证授权、数据访问、文件、日志、任务 |
| 部署配置 | `deploy` | 容器编排、反代、初始化、升级脚本 |

## 博客前台

| 模块 | 说明 |
| --- | --- |
| 首页 | 展示站点信息、轮播、推荐、最新文章、分类文章、说说入口 |
| 文章 | 文章详情、目录、上下篇、评论、阅读统计 |
| 归档 | 按时间线展示文章 |
| 分类/标签 | 按维度筛选文章 |
| 搜索 | 关键词搜索、搜索历史、分类/标签结果、AI 对话 |
| 说说 | 动态内容展示 |
| 相册/图床 | 图片资源展示 |
| 友链 | 友情链接展示 |
| 留言板 | 访客留言 |
| 用户中心 | 个人资料、绑定信息 |
| 效率工具 | 待办、任务池、日历、甘特图、日记、思考、时间块、习惯、AI 助手 |
| 布局与公共组件 | Header、Drawer、Footer、评论、登录弹窗、搜索弹窗、音乐播放器 |

## 后台管理

| 模块 | 说明 |
| --- | --- |
| 首页 | 后台概览数据 |
| 内容管理 | 文章、页面、分类、标签、说说、相册、照片、轮播 |
| 评论留言 | 评论、留言审核和管理 |
| 系统管理 | 用户、角色、菜单、权限、文件 |
| 日志监控 | 操作日志、异常日志、访问日志、任务日志、在线用户、定时任务 |
| 网站配置 | 站点信息、登录方式、社交链接、上传策略、音乐开关 |

## 后端服务

| 模块 | 说明 |
| --- | --- |
| 登录认证 | 登录、验证码、第三方登录、后台登录 |
| 博客信息 | 站点信息、关于页、后台概览 |
| 内容服务 | 文章、分类、标签、友链、说说、相册、照片、评论、留言 |
| 系统服务 | 用户、角色、菜单、文件、站点配置 |
| 日志服务 | 操作、异常、访问、任务日志 |
| 定时任务 | Quartz 任务、任务日志 |
| 效率工具 | 待办、任务池、日记、思考、时间块、习惯追踪 |
| AI 服务 | 聊天、提示词、总结、建议 |

## 跨模块约定

- 路由变化要同步需求文档和模块说明。
- 菜单、权限变化要同步后端菜单逻辑和后台前端说明。
- 接口返回结构变化要同步受影响的页面和模块说明。
- 站点配置变化要同步前台展示行为说明。
